# Backend emplate

This is a Node.js backend project.

## Getting Started

### Prerequisites

- Node.js
- npm

### Installation
1. Install the dependencies:
    ```bash
    npm install
    ```

### Running the Project

To start the development server, run:
```bash
npm start
```

### Running Tests

To run the tests, use:
```bash
npm test
```
